Architecture

Autonomous Agent Loop: LLM + File System + Shell Interface with structured action execution and context accumulation.

Core Components

  • LLM Interface: OpenAI API with typed response schemas (ResponseBody class)
  • File Handler: Read/write operations with absolute path management
  • Shell Interface: Command execution with 240s timeout protection
  • Context System: Linear actionresult chain with planned graph upgrade

Execution Model

python
while not finished:
    context_str = serialize(self.context)
    response = llm.generate(context_str, ResponseBody)
    
    if response.action == 0:  # File operation
        result = file_system.read/write(response.file_name, response.content)
    else:  # Shell command
        result = shell.execute(response.shell_command, timeout=240)
    
    self.context.append({"action": response, "result": result})

Key Technical Features

  • Typed Action Schema: Prevents malformed LLM responses
  • Timeout Protection: Kills hanging processes automatically
  • Context Persistence: Maintains state across multi-step operations
  • Environment Isolation: Works within virtual environments
  • Error Recovery: Continues execution after failures

Planned Enhancements

  • Merkle Tree Memory: Hierarchical context with hash-based summaries
  • Smart Pruning: Keep architectural decisions, summarize implementation details
